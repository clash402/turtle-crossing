# Turtle Crossing

## Summary
Quickly move the turtle to the other side without getting hit! [Try it out](https://repl.it/@clash402/turtle-crossing)

## Instructions
- Tap the Run button
- Move the turtle while avoiding the cars
- Have fun

## Controls
- Press <kbd>UP</kbd> to move up
